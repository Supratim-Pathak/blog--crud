## Getting Started

##### To run the application locally follow the steps below.

### Prerequisites

- **Docker CLI**
- **Docker Desktop application (Optional)**

### Installation

1. **Clone and Install App**:

- To clone the repository run the following command:

```bash
git clone https://github.com/Supratim-Pathak/blog--crud.git
```

- Get inside the project folder

```bash
cd blog--crud
```

- To install the dependencies of backend run the following command:

```bash
cd backend
npm install
```

- To install the dependencies of frontend run the following command:

```bash
cd frontend
npm install
```

2. **Run the Server**:

- To start the docker containers of backend and frontend run the following command in the root folder:

```bash
docker compose up --build
```

- To stop the docker containers of backend and frontend run the following command in the root folder:

```bash
docker compose down
```
